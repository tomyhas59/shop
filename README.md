# 쇼핑몰 프로젝트

쇼핑몰 프로젝트는 다양한 제품을 판매하고 구매할 수 있는 온라인 플랫폼입니다. 이 프로젝트는 최신 웹 기술을 사용하여 구축되었으며, 사용자에게 직관적인 인터페이스와 뛰어난 사용자 경험을 제공합니다.

## 기능

- 제품 목록 보기
- 제품 검색 및 필터링
- 장바구니에 제품 추가
- 주문 및 결제 처리
- 사용자 계정 관리 (회원가입, 로그인, 프로필 관리)
- 관리자 대시보드 (제품 관리, 주문 관리)
